pyOS
====

A fresh way to interact with your python objects as though they were files on your filesystem.

Get started
-----------

As easy as:

1. Install MongoDB

   Ubuntu::

    sudo apt install mongodb

2. Install pyos::

    pip install pyos


3. Dive in!::

    > ipython

    In [1]: from pyos.pyos import *
    In [2]: ls()
